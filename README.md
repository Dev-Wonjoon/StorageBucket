# StorageBucket

StorageBucket is a powerful desktop media downloader and manager based on `yt-dlp`. It provides an intuitive interface to save and organize media from various platforms without complex configurations.

---

## Language
- **English** (Current)
- [**한국어 (Korean)**](./README.ko.md)

---

## Key Features

* **yt-dlp Based Downloader**: Provides stable and high-speed downloads using the robust `yt-dlp` engine.
* **Wide Platform Support**: Supports numerous video platforms compatible with `yt-dlp`, including YouTube.
* **Gallery Management**: Systematically organizes and displays downloaded media in a clean gallery view for easy access.
* **Quality & Format Options**: Allows users to choose between video/audio types and various quality settings such as Best, 1080p, or 720p.
* **Modern UI**: Features a sleek, responsive, and dark-themed user interface built with React and Tailwind CSS.
