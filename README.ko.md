# StorageBucket

`yt-dlp`를 기반으로 다양한 플랫폼의 미디어(비디오/오디오)를 손쉽게 저장하고 관리할 수 있는 데스크톱 어플리케이션입니다. 복잡한 명령줄 설정 없이 직관적인 인터페이스를 통해 원하는 콘텐츠를 소장할 수 있습니다.

## 주요 기능

* **yt-dlp 기반 다운로더**: 강력한 `yt-dlp` 엔진을 탑재하여 고화질 비디오를 안정적이고 빠르게 다운로드합니다.
* **다양한 사이트 지원**: YouTube를 비롯하여 `yt-dlp`가 지원하는 수많은 비디오 플랫폼에서 미디어를 추출할 수 있습니다.
* **갤러리 관리**: 다운로드한 미디어를 데이터베이스화하여 리스트 형태로 한눈에 확인하고 체계적으로 관리합니다.
* **화질 및 포맷 선택**: 사용자의 필요에 따라 비디오 또는 오디오 타입을 선택할 수 있으며, Best, 1080p, 720p 등 다양한 화질 설정을 지원합니다.
* **현대적인 UI**: React와 Tailwind CSS로 구축된 세련된 다크 모드 기반의 사용자 인터페이스를 제공합니다.
