# StorageBucket

StorageBucket은 사용자가 관리하는 외부 다운로드 엔진을 연동해, 권한이 있는 미디어를 저장하고 정리할 수 있도록 돕는 데스크톱 미디어 라이브러리 관리 앱입니다.

이 앱은 사용자가 소유했거나, 다운로드 권한이 있거나, 다운로드가 허용된 라이선스로 제공되는 미디어를 관리하는 용도로 설계되었습니다.

## 주요 기능

* **외부 엔진 연동**: 사용자가 관리하는 `yt-dlp`, `gallery-dl`, `FFmpeg` 같은 외부 도구와 함께 사용할 수 있습니다.
* **권한 있는 미디어 관리**: 사용자가 소유했거나 다운로드 권한이 있는 미디어, 또는 다운로드가 허용된 라이선스의 미디어를 정리하는 데 초점을 둡니다.
* **갤러리 관리**: 저장한 미디어를 깔끔한 갤러리 형태로 정리하고 확인할 수 있습니다.
* **품질 및 형식 옵션**: 필요에 따라 비디오/오디오 유형과 품질 옵션을 선택할 수 있습니다.

## 책임 있는 사용

StorageBucket은 사용자가 소유했거나, 다운로드 권한이 있거나, 다운로드가 허용된 라이선스로 제공되는 미디어를 저장하고 관리하기 위한 도구입니다.

사용자는 관련 저작권법, 플랫폼 이용 약관, 콘텐츠 라이선스를 준수할 책임이 있습니다. StorageBucket은 DRM 우회 기능을 포함하지 않으며, 허가 없이 저작권 보호 콘텐츠에 접근하거나 다운로드하거나 배포하는 용도로 사용되어서는 안 됩니다.
